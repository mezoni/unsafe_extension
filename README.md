unsafe_extension
=====

Library for performing operations on the unmanaged native code and data.

Version: 0.0.24

**Installation**

Go to any application that uses `unsafe_extension` and run

```
$ pub global activate unsafe_extension
```

**Precompiled binaries**  

- ARM Android
- ARM Linux
- X86 Linux
- X86 Windows
- X86_64 Linux
- X86_64 Mac OS X
- X86_64 Windows
